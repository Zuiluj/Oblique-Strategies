# Oblique-Strategies
A program mainly for musicians who are stuck, but also for others if they see it as useful! 

Requires:
Python 3.7.2
PyQt5


1. Download 

2. Type in terminal:

    ```
    // change directory to folder
    cd dir/to/project
    
    // install requirements
    python pip install -r requirements.txt
    
    // Run the app
    python main.pyw
    ```
3. Enjoy new strategies
